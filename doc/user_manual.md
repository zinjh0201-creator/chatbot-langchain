# 사용자 매뉴얼 — RAG Chatbot

## 1. 시작하기
1. 브라우저에서 배포된 URL 접속 (예: https://chatbot01-he7o.vercel.app)
2. 상단 좌측에 있는 
   - 문서 업로드 패널 표시
   - 채팅 영역(우측) 표시
3. 질문 영역에 함수를 입력 후 전송

## 2. 1) PDF 업로드
- 드래그 앤 드롭 또는 `Browse files` 클릭 후 PDF 선택
- 최대 200MB까지 업로드 가능
- 업로드 성공 시 상태 메시지 없음, 실패 시 오류 표시

## 3. 2) 질문 보내기
1. 텍스트 상자에 질문 입력
2. Enter (Shift+Enter 아님) 또는 전송 버튼 클릭
3. 입력 시 텍스트가 즉시 채팅에 추가
4. 봇 응답까지 대기

## 4. 상태 안내
- `▶` 로딩 표시: 질문 응답 대기 중
- `PDF 업로드 실패` / `업로드 중` 문구 표시

## 5. 엣지 케이스
- 메시지 공백 입력: 전송 불가
- 서버 연결 문제: `[오류]`로 응답 표시
- 질문 응답 지연: 10초 이상이면 네트워크/백엔드 확인

## 6. 데이터 보존
- 대화 내용은 로컬스토리지에 자동 저장
- 새로고침 시 복원됨
- `히스토리 삭제` 버튼으로 초기화

## 7. 문제 해결
1. 화면이 하얀 경우
   - 브라우저 개발자 도구(F12) 콘솔 확인
   - 네트워크 `index.js`, `style.css` 로드 확인
2. API 오류
   - `VITE_API_BASE` / 백엔드 URL 환경 변수 확인
3. PDF 파싱 오류
   - PDF에 텍스트가 있는지 확인

## 8. 관리자 모드(운영 가이드)
- 백엔드 `.env`:
  - `DB_URL` (Supabase 연결 문자열, `?sslmode=require` 포함)
  - `GEMINI_API_KEY`
- 배포시 `Vercel` 환경변수:
  - `VITE_API_BASE` 설정
  - 백엔드 실제 URL
